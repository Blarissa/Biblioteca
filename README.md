# Biblioteca
**Descrição do projeto**

Aplicação console para um repositório de livros

**Funcionalidades**

**Livro**
- Adicionar livro
- Listar todos os livros
- Pesquisar livro

**Leitor**
- Adicionar leitor
- Listar todos os leitores
- Pesquisar leitor

**Empréstimo**
- Adicionar espréstimo
- Listar todos os espréstimos
- Pesquisar espréstimo
